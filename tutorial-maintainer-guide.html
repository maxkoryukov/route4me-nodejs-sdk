Dev tools

We use GULP as an task-automation tool. It has several useful development task, described below.

lint

Provides linting for *.js sources

gulp lint

One useful feature: autofix style-issues. You could significantly reduce the amount of the manual fixes with this command:

gulp lint --fix

test

Run mocha tests

gulp test

#or with grep:
gulp test --grep=integration
